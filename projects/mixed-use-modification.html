Modification of Existing Mixed-Use Building

Student Project

Redesign of an existing mixed-use building while maintaining its original programs and functions, focusing on improving functionality, user experience, and spatial efficiency.

Project Aim

The aim of this project was to enhance and modernize an existing mixed-use building without disrupting its current program, ensuring improved circulation, usability, and visual appeal.

Key Focus

  • Maintaining original building functions
  • Spatial optimization
  • Improved user flow
  • Modernized design elements

Skills & Tools

  • 3D Modeling
  • Rendering
  • Architectural Design
  • Presentation
